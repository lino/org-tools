#+TITLE: org-tools — Installation

* From Source

Requires Rust 1.83+ (or use =mise= to install the pinned version).

#+begin_src sh
git clone https://github.com/org-tools/org-tools.git
cd org-tools
cargo install --path crates/org
#+end_src

Or with mise:

#+begin_src sh
mise install        # installs pinned Rust version
mise run build      # cargo build
#+end_src

* From Cargo

#+begin_src sh
cargo install org
#+end_src

* Shell Completions

Generate and install shell completions:

#+begin_src sh
# Bash
org --completions bash > /etc/bash_completion.d/org

# Zsh
org --completions zsh > ~/.zsh/completions/_org

# Fish
org --completions fish > ~/.config/fish/completions/org.fish
#+end_src

* Verify Installation

#+begin_src sh
org --help
org fmt check --help
org query search --help
org clock report --help
org export ical --help
org update set-state --help
org archive --help
#+end_src
