#+TITLE: org-tools — Installation
#+DATE: 2026-03-21

* From Source

Requires Rust 1.70+ (or use =mise= to install the pinned version).

#+begin_src sh
git clone https://github.com/YOUR_USER/org-tools.git
cd org-tools
cargo install --path crates/org
#+end_src

Or with mise:

#+begin_src sh
mise install        # installs pinned Rust version
mise run build      # cargo build
#+end_src

* From Cargo

#+begin_src sh
cargo install org
#+end_src

* Verify Installation

#+begin_src sh
org --help
org fmt check --help
org fmt format --help
org query search --help
org query agenda --help
#+end_src
